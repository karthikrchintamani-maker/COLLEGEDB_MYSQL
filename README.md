# 🎓 College Database Management System (Node.js + MySQL)

## 📌 Project Overview

The College Database Management System is a backend application developed using Node.js, Express.js, and MySQL. It performs database operations such as creating a college database, creating tables, inserting records, and retrieving student information.

---

## 🚀 Features

- Create College Database
- Create Student Table
- Insert Student Records
- Retrieve Student Details
- MySQL Database Integration
- REST API using Express.js

---

## 🛠 Technologies Used

- Node.js
- Express.js
- MySQL
- mysql2
- JavaScript

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/karthikrchintamani-maker/COLLEGEDB_MYSQL.git
```

Go to the project directory

```bash
cd COLLEGEDB_MYSQL
```

Install dependencies

```bash
npm install
```

---

## ▶ Run the Project

```bash
node index.js
```

Server runs on

```
http://localhost:8000
```
